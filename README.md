# Sai Kiran's Portfolio

## About

Personal portfolio showcasing machine learning, data science, and web development projects.

## Projects

- **Spam Email Detector** — NLP-based spam classifier using Naive Bayes (~98% accuracy)
- **Digit Recognizer CNN** — CNN on MNIST with 99%+ accuracy and interactive canvas
- **Score Predictor** — Regression model predicting student exam scores with full EDA
- **Terminal Task Manager** — CLI productivity tool with priorities and deadlines
- **COVID-19 Dashboard** — Interactive charts on global COVID trends

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Recharts for data visualization
